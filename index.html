Questão 1

Crie uma função chamada sortArrayStr(arr,typeSort), que receba, como parâmetro, um vetor de palavras (arr) e a forma de ordenação (typeSort). A forma de ordenação pode ser Crescente (typeSort=1) ou Decrescente (typeSort=-1). O vetor deve ter no máximo 100 posições, caso contrário, a função retornará "null". Caso o valor de typeSort seja diferente de um ou menos um, a função deve também retornar "null". Caso os parâmetros estejam corretos, a função deve retornar a um vetor ordenado. (1 ponto)

Questão 2

Faça uma função chamada searchItems(arr,key,value), que recebe como parâmetros um Array de Objetos Literais (arr), uma chave (key) e um valor para buscar (value). A função deve procurar em cada objeto armazenado no vetor, uma key (chave) que contenha um valor (value), fornecido. Nos Objetos Literais armazenados no vetor, deve existir as chaves "nome", "idade" e "data-nascimento". A primeira chave deve receber valores do tipo String; a idade, Number; e a última, Date. Caso a key fornecida como parâmetro não esteja presente nos objetos fornecidos, a função deve retornar o tipo especial "undefined". Caso o valor fornecido no parâmetro value não esteja presente nos objetos, deve ser retornando "null". Se houver objetos com o valor procurado, deve-se retornar um vetor que contenha esses objetos. (1 ponto)

Questão 6

Faça uma função chamada listPots(url), que recebe como parâmetro um URL para uma função remota que retorna todos os comentários de armazenados em um servidor. A função retorna um Array de objetos com os dados fornecidos ou, caso não seja retornado nada, ela retorna o valor especial "null". Depois, faça uma aplicação web que acesse a função remota presente no endereço https://jsonplaceholder.typicode.com/posts e liste e pagine os dados recuperados. Use sua função para acessar os dados retornados pela função remota. Os dados retornados são "comentários de usuários", logo, você deve criar uma interface que permite ao utilizador acessar esses dados de forma compreensível e agradável. (1 ponto)

Questão 7

Crie um novo projeto, aproveitando a função listPots(url), descrita no exercício anterior, modifique-a e crie uma função chamada listPotsByUset(url,userid) que retorne somente as mensagens pertencentes a um usuário identificado pelo parâmetro userid. Modifique, também, a interface de exibição das mensagens paginadas a fim de que o utilizador possa filtrar as mensagens exibidas, por usuário que a criou. Como os dados retornados não possuem o nome dos usuários que criaram as mensagens, faça o filtro usar somente o valor do identificador de usuário, que seria selecionado pelo utilizador a gim de fazer a filtragem. Lembre de usar o endereço https://jsonplaceholder.typicode.com/posts para acessar as mensagens. (1 ponto)
