Questão 3

Em CSS, um Seletor define a quais elementos um conjunto de regras (rules) se aplica. Ele pode ser o descritor de uma tag, o identificador (Id) de um elemento ou a classe (class) a qual está associada um elemento. O CSS permite a combinação de seletores a fim de definir regras que se apliquem mais precisamente a um conjunto de elementos desejados. Um “Combinador” (Combinator) é um elemento que permite esse processo. Quais opções abaixo possuem Combinadores do tipo Child Combinator, General Sibling Combinator ou Adjacent Sibling Combinator? (Para pesquisar) (1 pto)

A) div ~ p { … }

Este é um exemplo do General Sibling Combinator (~)

B) p || ul < p { … }

Esta sintaxe não é válida

C) div + ul > li span { … }

Este é um exemplo do Adjacent Sibling Combinator (+)

D) div > p + span { … }

Este é um exemplo do Child Combinator (>)

Child Combinator (>): Este combinador seleciona elementos que são filhos diretos do elemento à esquerda.

General Sibling Combinator (~): Este combinador seleciona elementos que são irmãos no mesmo nível do elemento à esquerda. Não precisa ser necessariamente um irmão direto, mas precisa estar no mesmo nível da hierarquia.

Adjacent Sibling Combinator (+): Este combinador seleciona elementos que são irmãos no mesmo nível do elemento à esquerda e que estão imediatamente após o elemento à esquerda.
